# pkgctl auth

> Authenticate `pkgctl` with services like GitLab.
> More information: <https://manned.org/pkgctl-auth.1>.

- Authenticate `pkgctl` with the GitLab instance:

`pkgctl auth login`

- View authentication status:

`pkgctl auth status`
