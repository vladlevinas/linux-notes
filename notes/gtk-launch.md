# gtk-launch

> Launch applications from `.desktop` files that reside in standard locations.
> More information: <https://manned.org/gtk-launch>.

- Launch an application:

`gtk-launch {{Application_name}}`

- Display help:

`gtk-launch {{[-h|--help]}}`
