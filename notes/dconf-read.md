# dconf read

> Read key values from dconf databases.
> See also: `dconf`.
> More information: <https://manned.org/dconf>.

- Print a specific key value:

`dconf read /{{path/to/key}}`

- Print a specific key [d]efault value:

`dconf read -d /{{path/to/key}}`
