# Linux Notes (TLDR → hourly notes)

- В папке `vendor/tldr/` хранится офлайн-копия TLDR страниц (Linux + common).
- Каждый час из этого набора переносится **одна** страница в `notes/` и коммитится.
- Индекс заметок: `notes/README.md`.

## Источник TLDR
